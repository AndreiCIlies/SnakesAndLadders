# Snakes and Ladders Game

## Description

Snakes and Ladders is a Java-based board game built using the Greenfoot framework.  
This project simulates the classic game where two players take turns rolling a dice and advancing across a 10×10 board from START to FINISH. Snakes move players backward, while ladders help them climb forward.

The game includes:

- **Two-player turn-based system (Red and Green players)**
- **Interactive clickable dice**
- **Automatic snake and ladder jumps**
- **Win detection and board-centered victory message**
- **Graphical board with tile numbering, START, and FINISH labels**
- **Current turn and rolled value displaying**

## Technologies Used

- **Java**
- **Greenfoot**

## Installation and Usage

### Prerequisites

- Greenfoot

### Installation

**1. Clone the repository**  
git clone https://github.com/AndreiCIlies/Snakes_And_Ladders.git

**2. Open the project in Greenfoot**

Launch Greenfoot

Go to Project → Open

Select the folder containing project.greenfoot

**3. Run the game**
